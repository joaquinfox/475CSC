Great post Aaron. I especially like your discussion of string extraction and your argument for considering that a best practice.  To flesh that out in some detail, we could implement a string extraction as follows:

  • Instead of assigning a string in XML,
    android.id = "Long, long time ago ..."
    We assign an id ...
    android.id = "@+id/someUniqueIdentifier"
  •  Then, we can define a string variable in XML and put it in our resource folder.
  • Now we can call that string value from within the layout XML 
    
    android:text="@string/firstString"
